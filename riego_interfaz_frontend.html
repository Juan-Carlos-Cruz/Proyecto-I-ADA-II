Algoritmos
Escoge la estrategia de calculo
Cada opcion resalta un compromiso distinto entre costo computacional, velocidad y garantia de optimalidad.
💪
Exhaustivo
Fuerza Bruta
Prueba todos los ordenes posibles y sirve como referencia exacta cuando `n` es pequeno.
⚡
Rapido
Voraz
Toma decisiones paso a paso para construir una solucion agil con bajo costo de ejecucion.
🧠
Optimo
Programacion Dinamica
Busca una solucion optima con mejor control del espacio de estados que la fuerza bruta.
Fuerza bruta con `n` grande puede tardar mucho. Se recomienda principalmente para `n <= 12`.
